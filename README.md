# NavMesh Diploma Demo

Демонстрационная сцена для магистерской работы:
«Исследование алгоритмов навигации NPC в игровых сценах с учётом стоимостей маршрута и динамических препятствий».

## Сцена

Основная сцена находится в `Assets/Scenes/NavMeshDemo.unity`.

В сцене есть:

- плоская игровая область со стенами;
- стартовая точка и цель;
- несколько маршрутов от старта к цели;
- короткий проход через зону `Expensive`;
- более длинные альтернативные обходы;
- `NavMeshSurface` из пакета AI Navigation;
- NPC на `Capsule` с `NavMeshAgent`;
- динамическое препятствие с `NavMeshObstacle` и carving;
- экранные метрики через `OnGUI`.

Если сцену нужно пересобрать, в Unity доступен пункт меню:
`Tools/NavMesh Diploma Demo/Rebuild Scene`.

## Управление

- `Space` - старт движения активных NPC.
- `R` - сброс эксперимента.
- `C` - включить/выключить повышенную стоимость зоны `Expensive`.
- `O` - включить/выключить динамическое препятствие.
- `N` - переключить один NPC / несколько NPC.

## Метрики

На экране выводятся:

- длина текущего построенного пути;
- время достижения цели;
- количество перестроений маршрута;
- FPS;
- количество активных агентов.

## Эксперименты

1. Кратчайший путь:
   - нажать `R`, затем `Space`;
   - зона `Expensive` имеет обычную стоимость;
   - агент выбирает короткий центральный проход.

2. Учёт стоимости зоны:
   - нажать `C`, затем `R` и `Space`;
   - стоимость `Expensive` повышается;
   - агент должен предпочесть более длинный обход, если суммарная стоимость маршрута ниже.

3. Динамическое препятствие:
   - нажать `O` до старта или во время движения;
   - препятствие перекрывает центральный проход;
   - агент перестраивает маршрут вокруг препятствия.

4. Несколько NPC:
   - нажать `N`, затем `R` и `Space`;
   - активируются несколько агентов;
   - можно наблюдать избегание друг друга и сравнить метрики.

## Основные скрипты

- `Assets/Scripts/NavMeshDemoController.cs` - управление режимами, вводом, стоимостью зон, препятствием, сбросом и UI.
- `Assets/Scripts/NavMeshDemoAgent.cs` - запуск агента, измерение пути, времени и перестроений.
- `Assets/Scripts/NavMeshDemoPathRenderer.cs` - отрисовка текущего пути агента.
- `Assets/Editor/NavMeshDemoSceneBuilder.cs` - автоматическая сборка демонстрационной сцены.
